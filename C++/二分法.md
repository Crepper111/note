# 二分法

有一个序列，这个序列的前一段都满足一个条件，后一段都不满足这个条件，二分法可以快速求出这个序列关于这个条件的**零点**。

- 时间复杂度 $O(logn * check)$

- $n$ 是值域， $check$ 是判断条件的复杂度

## code：
```cpp
int l=min,r=max;
while(l<r)
{
	int mid=(l+r)/2;
	if(check(mid)) l=mid+1;
	else r=mid;
}
ans=l;
```

```cpp
int l=min,r=max;
while(l<r)
{
	int mid=(l+r+1)/2;
	if(check(mid)) l=mid;
	else r=mid-1;
}
ans=l;
```
两份代码的区别：前者求的是第一个不能满足条件的值的下标，后者求的是最后一个满足条件的下标

### 注意：

- 最终答案都是l

- 边界可以开多，但不能开小。
